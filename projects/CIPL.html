custom rtl ip library

overview

This project is a growing custom RTL IP library focused on designing reusable, parameterized SystemVerilog components for processor datapaths and SoC-style systems. The library emphasizes clean interfaces, hardware-realistic implementations, and composability, with IP blocks designed to reflect patterns used in real microarchitectures rather than single-purpose designs.

Current components include an RV32-style integer ALU, synchronous and asynchronous FIFOs for buffering and clock-domain crossing, and configurable arbiters supporting round-robin and fixed-priority policies.
