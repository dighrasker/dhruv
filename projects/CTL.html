custom thread library

overview

This project implements a user-level thread library in C++ that supports preemptive and cooperative multithreading across single- and multi-CPU environments, closely modeling core operating-system scheduling and synchronization mechanisms. The library provides thread creation, context switching, joining, mutexes, and condition variables, with execution driven by timer and inter-processor interrupts and coordinated through a shared FIFO-ready scheduler.

Thread execution is managed using x86-64 ucontext primitives (makecontext, swapcontext, setcontext), with careful handling of thread lifetimes, stack allocation, and atomic interrupt control to ensure correctness under concurrency.

The implementation emphasizes OS-level invariants such as atomic context switches, efficient CPU suspension when idle, FIFO fairness, and robust error detection, offering hands-on experience with low-level scheduling, synchronization, and multiprocessor coordination.
